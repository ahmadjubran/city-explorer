**Author**: Ahmad Jubran

## Overview

City explorer is a web application that allows users to explore the cities of the world.

## Getting Started

    1. Clone the repository
    2. Install dependencies
    3. Run the server
    4. Open the browser and navigate to http://localhost:3000

## Architecture

- HTML and CSS
- JavaScript
- React
- React Bootstrap
- LocationIQ API
